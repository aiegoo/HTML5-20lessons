2. Curves

arc() Function

arc() function takes 6 arguments.

  • A pair of coordinates representing the center of circle upon which the arc is based.
  • The Radius
  • The Radian offset value for the start of the arc
  • The radian offset value for the end of the arc
  • value indicating whether to draw the arc clockwise or counterclockwise

The quadraticCurveTo() function

you can create the most fancy of curves using this function. which employs an imaginary attactor object that pulls the curve towards ot for example : let draw a line between left and right side of canvas using simple stroke() call.

context.beginPath()
context.moveTo(0, 85)
context.lineTo(410, 85)
context.stroke()
context.closePath()

now let draw a curved line between thses positions but with an imaginary attactor up in the top left hand corner at location 0,0 :
context.beginPath()
context.moveTo(0, 85)
context.quadraticCurveTo(0,0,410,85)
context.stroke()
context.closePath()
