Conditional HTML for Internet Explorer

you can use IE conditional comments by inserting a pair of square brackets immediately following the opening comment marker, placing an if statement inside.

Simple Comparisons

IE constant is tested to see whether it has a value of 6, indicating that the current browser is IE6. you can also simply test for browser being any version of IE by leaving out the numeric value.

Higher or Lower Values

if you wish to display some HTML only to versions of Internet Explorer prior to version 9, you can use this form of conditional HTML. lt for less than , lte for less than equal to, gt for greater than, gte for greater than equal to.

The NOT operator

you can also test the inverse of a comparison using the not operator(!), which displays the contents only if the browser is Internet Explorer , but not version 6

The Mark of the Web

Internet Explorer also uses comments to stamp what it calls the Mark of the Web onto an HTML document, as a way of setting the security zone to which a document applies during the development.

Base Tag

Use this tag to specify the base destination for all URLs in a document. for example- if you wish all relative links in a document to refer to the base URL https://mywebsite.com/project/ even if the document is located else where ont he internet.

now, any hyperlink that are relative will be applied to that base. for example the following will now link to https://mywebsite.com/project/. It will work if you serve the current document from a local file system or from anywhere else making this a great way to handle documents that have to be relocated away from their original location for some reason.
