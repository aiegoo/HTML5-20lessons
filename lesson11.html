1. Writing Text

The font Property

  • Example - context.font= '30pt Arial
  • Font Size Measurement:
    1. em- based on default preferences set in the browser
    2. ex- X-height: based on height of lowercase x character
    3. px- Pixels: based on resolution of screen
    4. % - Percentage:based on default preference of the browser
  • Fixed Unit are:
    1. in- Inches
    2. cm - centimeters
    3. mm - Millimeters
    4. pt- Points: a print unit
    5. pc- Picas: Another point unit
  • Examples:
    1. context.font= '3em Helvetica'
    2. context.font= '16px Impact'
    3. context.font= '150% Courier'
    4. context.font= '10mm Times'
    5. context.font= '72pt Arial'
