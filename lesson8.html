2. Geolocation

  • it is technology that is used to determine the location of a computer or mobile devices, which can be returned to the web server in order to provide relevant information.
  • a local map can be displayed, or details of local stores such as restaurants or gas stations can be provided.
  • used to help improve web connectivity by notifying you of nearby wifi access points.
  • different methods are used to obtain location starting with IP address
  • With HTML5 geolocation, if WIFI is enabled on your computer, it is also possible to send more precise location information by scanning your locally accessible WIFI hostpots to pinpoint the location
  • if your computer has GPS functionality, this can reveal your exact location to within just a few feet, as long as your device is in range of the GPS satellites that orbit the earth

3. Forms

  • Forms have been provided with a number of new enhancements in HTML5
  • you can place input tag outside of form and /form tags as long as the new form attribute is used to identify the form ID to which the input refers.
  • you can change the method- get or post of the form with new formmethod attribute.
  • you can chnage encoding type of a form or create or override the new formnovalidate attribute.
  • you can also use formaction attribute to change the action of the form
  • you can use the formtarget attribute to change the target frame, tab or window.
  • you can change the width and height of image type of input tag using height and width attributes.
  • Data Lists can also be included in a form so that you can create a list of labels and values to which you assign an ID.Then you can use that ID as a value for the new list attribute to easily offer a selection of choices to an input.
  • New min, max and step attributes have also been added for inputs that contains numbers or dates, and there's new multiple attribute with which you can allow an input tag to accept multiple values.
  • Pattern matching is supported in forms
  • several new values are now supported for the type attribute, including color, date, month, time,week,number,range,tel,url and email.

It is subset of HTML designed for making a document have meaning to machines by providing metadata. just as it has meaning to a reader of the document.
