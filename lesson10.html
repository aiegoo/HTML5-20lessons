1. Drawing Rectangles

The fillRect() function

  • it takes 4 arguments reprsenting - vertical and horizontal offsets of the top-left hand corner of a rectangle from the top-left corner of the canvas followed by the rectangle width and height all of which are in units of pixels or one screen dot.
  • following syntax create a filled rectangle 50 pixels wide and 70 pixels high , an offset of 20 pixels in from left egde of canvas (410/170 pixels) and 30 pixels down from top.

The fillstyle Property

  • you can set the type of fill color to use using hexadecimal digits
  • example1: context.fillStyle = '#F00'
  • example2: context.fillStyle = 'red'

The clearRect() function

  • if you want to draw a clear rectangle in which all the RGBA values of pixel are set to zero, you can use clearRect() function.
  • example1: context.clearRect(20,30,50,70)
  • the cleared area is stripped of all colors, leaving only underlying background color that may be applied to canvas tag.

The strokeRect() Function

  • you can create rectangle that uses the current strokeStyle, lineWidth,lineJoin and miterLimit properties to draw the boredr lines and corners
  • example: context.strokeRect(20,30,50,70)

Using Patterns

The createPattern() Function

To create a pattern , you need to supply a pre-existing image such as a jpeg,png or gif file, and the type of repetition to use when applying the fill, out of following options:

  • repeat = repeat the image both vertically and horizontally
  • repeat-x = repeat the image horizontally
  • repeat -y = repeat the image vertically
  • no-repeat = donot repeat the image
